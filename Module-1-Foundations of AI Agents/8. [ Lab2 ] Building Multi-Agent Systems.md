## ğŸ¤– Multi-Agent System Overview

The system consists of **four agents**, each with a clearly defined role:

1. **Research Planner**
   - Analyzes the user query
   - Breaks it down into smaller, focused research topics
   - Produces a structured research plan

2. **Internet Researcher**
   - Executes the research plan
   - Uses web search and scraping tools
   - Collects detailed information from online sources

3. **Fact Checker**
   - Verifies claims, features, and pricing
   - Cross-checks sources for accuracy
   - Ensures reliability of gathered data

4. **Report Writer**
   - Synthesizes validated research
   - Produces a polished **Markdown report**
   - Includes summaries, tables, sources, and comparisons

## ğŸ§± Core CrewAI Concepts Applied

### 1. Agents
- Each agent has:
  - A specific role
  - A clear goal
  - A defined backstory
- Some agents (researcher, fact checker) share the same toolset

---

### 2. Tasks
- Tasks are **explicit and structured**
- Each task defines:
  - Objectives
  - Expected outputs
  - Quality constraints (e.g., comprehensive, well-sourced)

#### Variable Interpolation
- User input (the research query) is injected into tasks
- This input drives the entire workflow
- Enables dynamic reuse of the same crew for different topics

---

### 3. Sequential Execution

The crew operates **sequentially**, where each step builds on the previous one:

1. Research planning  
2. Information gathering  
3. Fact verification  
4. Report writing  

This ensures logical flow and higher reliability.
